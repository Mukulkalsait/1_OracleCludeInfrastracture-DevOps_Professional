
| Concept             | Real-world analogy                                                           | Example            |
| ------------------- | ---------------------------------------------------------------------------- | ------------------ |
| **Pod name**        | A specific *car’s license plate*                                             | `myapp-pod-1`      |
| **Container name**  | The *engine name* inside that car                                            | `backend-engine`   |
| **Label**           | A *sticker* saying “This is for delivery service”                            | `app=myapp`        |
| **Deployment name** | The *fleet manager* who manages many cars                                    | `myapp-deployment` |
| **Selector**        | The *rule* the manager uses to pick which cars belong to it                  | `app=myapp`        |
| **Service name**    | The *dispatcher’s phone number* that forwards calls to all cars in the fleet | `myapp-service`    |


| Field                        | Scope                   | Must be unique? | Purpose                                |
| ---------------------------- | ----------------------- | --------------- | -------------------------------------- |
| `metadata.name` (Pod)        | Inside a namespace      | ✅ Yes           | Identifies this specific Pod           |
| `metadata.name` (Deployment) | Inside a namespace      | ✅ Yes           | Identifies the controller              |
| `containers[].name`          | Inside that Pod         | ✅ Yes           | Used by probes, logs, etc.             |
| `labels`                     | Any level               | ❌ No            | Used to *group* and *select* resources |
| `annotations`                | Any level               | ❌ No            | Metadata for humans/tools              |
| `selector.matchLabels`       | In Deployment/Service   | ❌ No            | Filters resources by label             |
| `envFrom.configMapRef.name`  | Global within namespace | ✅ Yes           | Refers to ConfigMap name               |
